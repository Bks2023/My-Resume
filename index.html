LVRT Replacement Using Deep Reinforcement Learning

I applied deep reinforcement learning to replace conventional LVRT mechanisms, enhancing system performance during voltage sags. I developed and tested models in software environments to simulate and optimize grid stability under fault conditions, using Typhoon for simulation. AI-based LVRT solutions were integrated into hardware systems for physical testing, validating their effectiveness in real-world power environments. This approach demonstrated improved voltage ride-through capabilities and significantly reduced system downtime during low voltage scenarios.

Steganographic image transmission using BPSK modulation

I implemented steganography to hide secret information in images, utilizing techniques such as Least Significant Bit (LSB) and Pixel Value Differencing (PVD). The process involved converting images into binary digits, embedding secret messages, and transferring data using BPSK modulation. I used the Dream Catcher training kit for digital communication system development, combining steganography, encryption, and Dream Catcher transmission technology to ensure secure communication.

This project focuses on setting up a virtualization environment using Oracle Linux 8.7, oVirt, and KVM, supported by Intel® Xeon® processors and DDR3 RAM.
